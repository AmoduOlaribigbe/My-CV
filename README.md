
  # Portfolio website for CV

  This is a code bundle for Portfolio website for CV. The original project is available at https://www.figma.com/design/E3egieLwbXvf7lBjzbzMJf/Portfolio-website-for-CV.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deploying to GitHub Pages

  1. Make sure your `vite.config.ts` has the correct `base` property (e.g., `/My-CV/`).
  2. Build the project:
    ```bash
    npm run build
    ```
  3. Deploy to GitHub Pages:
    ```bash
    npm run deploy
    ```
  4. Your site will be available at: https://AmoduOlaribigbe.github.io/My-CV
